Origem

O Arduino é uma plataforma de prototipagem eletrônica de código aberto que teve origem na Itália. Foi criado por Massimo Banzi

Como usar

Usar o Arduino envolve alguns passos básicos, desde a configuração do ambiente de desenvolvimento até a escrita e upload de código para a placa

O que é ?

O Arduino é uma plataforma de prototipagem eletrônica de código aberto que permite criar projetos interativos
